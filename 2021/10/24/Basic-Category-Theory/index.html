Basic Category Theory

Categories

Definition of a category

Definition 1.   A catagory $\boldsymbol{C}$ consists of the following data:

  • a class $ob(\boldsymbol{C})$ of objects.
  • a class $hom(\boldsymbol{C})$ of morphisms.
  • for each $f\in hom(\boldsymbol{C})$, there is given objects $dom(f)$ and $cod(f)$, called the domain and codomain of $f$. We write $f\colon\ A\to B$ to indicate that $A=dom(f)$ and $B=cod(f)$.
  • given morphisms $f\colon\ A\to B,\ g\colon\ B\to C$, there is given a morphism $g\circ f\colon\ A\to C$, which is called the composition of $f$ and $g$.
  • for each $A\in ob(\boldsymbol{C})$, there is a morphism $1_A\colon\ A\to A$, called the identity morphism of $A$.

which satisfies the following laws:

  • Associativity:
    $$h\circ(g\circ f)=(h\circ g)\circ f$$
    for each $f\colon\ A\to B,\ g\colon\ B\to C,\ h\colon\ C\to D$.
  • Unit:
    $$f\circ 1_A=f=1_B\circ f$$
    for each $f\colon\ A\to B$.

Isomorphisms

Definition 2.   A morphism $f\colon\ A\to B$ is called an isomorphism, if there is a morphism $g\colon\ B\to A$, such that
$$g\circ f=1_A\quad f\circ g=1_B$$

Let $f\colon\ A\to B$ be an isomorphsm, if there are two morphisms $g,g’$ satisfy the property above, we have:
$$g=g\circ 1_B=g\circ(f\circ g’)=(g\circ f)\circ g’=1_A\circ g’=g’$$
Then $g$ is unique, we call $g$ is the inverse of $f$ and write $g=f^{-1}$. We say $A$ is isomorphic to $B$, written $A\cong B$, if there exists an isomorphism between them.
