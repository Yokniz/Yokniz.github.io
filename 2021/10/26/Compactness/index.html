Compactness

Compact, sequentially compact and limit point compact

Definition 1.   For a topological space $X$:

  • $X$ is said to be compact if each open covering of $X$ has a finite subcovering.
  • $X$ is said to be sequentially compact if each sequence of points of $X$ has a convergent subsequence converging to a point in $X$.
  • $X$ is said to be limit point compact if each infinite subset of $X$ has a limit point in $X$.

Proposition 1.   For a topological space $X$:

  1. If $X$ is compact, $X$ is limit point compact.
  2. If $X$ is sequentially compact, $X$ is limit point compact.

Proof.  

  1. Let $A$ be an infinite subset of $X$. If $A$ has no limit point, then $A$ is closed in $X$, and for each point $a\in A$, there is an open neighborhood $U_a$ of $a$, such that $U_a\cap A={a}$, then $A^\mathsf{c}\cup\{U_a\}_{a\in A}$ is an open covering of $A$. Since $X$ is compact, we have a finite covering $\{U_{a_1},U_{a_2},\dots,U_{a_n},A^\mathsf{c}\}$ of $X$, then $A=A\cap X=\bigcup_{i=1}^n (A\cap U_{a_i})=\{a_1,a_2,\dots,a_n\}$, which is a finite set. Contradicts.
  2. Since $A$ is an infinite set, we can find a sequence $\{x_n\}$ of points in $A$, where $x_i\neq x_j$ if $i\neq j$. Since $X$ is sequentially compact, we have a convergent subsequence $\{x_{n_k}\}$, suppose $\lim_{k\to\infty}x_{n_k}= x\in X$, then $x$ is a limit point of $A$ in $X$. $\square$

We say that $A\subset X$ is a compact set if the subspace $A$ is a compact space, same for sequentially compact and limit point compact.

Proposition 2.   $A\subset X$ is a compact set if and only if each open covering of $A$ in $X$ has a finite subcovering.

Definition 2.   A collection $\mathcal{F}=\{F_\alpha\}$ of subsets of $X$ is said to satisfy the finite intersection property, if each finite intersection $\bigcap_{i=1}^n F_{\alpha_i} \neq\varnothing$.

Proposition 3.   A topological space $X$ is compact if and only if for each collection $\mathcal{F}=\{F_\alpha\}$ of subsets of $X$ satisfies the finite intersection property, we have $\bigcap_\alpha U_\alpha\neq\varnothing$.

Corollary 1.   If $X$ is a compact space, $F_1\supset F_2\supset\dots$ is a non-increasing sequence of non-empty subsets of $x$, then $\bigcap_{i=1}^n F_i\neq\varnothing$.

Proposition 4.   A topological space $X$ is compact if and only if each basis covering of $X$ has a finite subcovering.

According to the Alexander subbase theorem, the term “basis” above can be replaced by “subbasis”.

Proposition 5.   $X$ and $Y$ are topological spaces, $f\colon\ X\to Y$ is continuous.

  • If $A$ is a compact subset of $X$, then $f(A)$ is a compact subset of $Y$.
  • If $A$ is a sequentially compact subset of $X$, then $f(A)$ is a sequentially compact subset of $Y$.

If $A$ is a limit point compact subset of $X$, the property above may not hold.

Proposition 6.   Compact subsets of Hausdorff spaces are closed.

Proposition 7.   Closed subsets of compact spaces are compact.

The duality of compactness and Hausdorff.

The compact Hausdorff spaces

Lemma 1.   If $\mathcal{T}_1\subset\mathcal{T}_2$ are two topologies on $X$, $(X,\mathcal{T}_1)$ is Hausdorff and $(X,\mathcal{T}_2)$ is compact, then $\mathcal{T}_1=\mathcal{T}_2$.

Proposition 8.   If $(X,\mathcal{T})$ is compact and Hausdorff, $\mathcal{T}_1,\mathcal{T}_2$ are two other topologies satisfy $\mathcal{T}_1\subsetneq\mathcal{T}\subsetneq\mathcal{T}_2$, then $(X,\mathcal{T}_1)$ is not Hausdorff and $(X,\mathcal{T}_2)$ is not compact. (Rigidity)

Compactness in metric spaces

Definition 3.   In a metric space, a subset $S$ is said to be bounded if $diam(A)=\sup\{d(x,y)\colon\ x,y\in A\}<+\infty$.

Proposition 9.   Compact subsets of metric spaces are bounded closed sets.

Example 1.   A bounded closed set of a metric space may not be compact. Consider the subset $\mathbb{N}\subset (\mathbb{R},d_{\text{discrete}})$ is bounded and closed, but not compact, where $d_{\text{discrete}}$ denote the following metric on $\mathbb{R}$:
$$
d(x,y)=
\begin{cases}
0 &x=y\\
1 &x\neq y
\end{cases}
$$

Proposition 9.   In metric spaces, the three kinds of compactness are equivalent.

Definition 4.   A metric space is said to be totally bounded if each of its $\epsilon$-ball coverings has finite subcovering.

Definition 5.   A metric space is said to be complete if each cauchy sequence of points in the space converges.

Lebesgue’s number lemma.  

Proposition 10.   If $X$ is a metric space, then $X$ is compact if and only if $X$ is totally bounded and complete.
