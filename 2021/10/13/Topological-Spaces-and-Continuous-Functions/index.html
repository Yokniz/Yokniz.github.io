Topological Spaces and Continuous Functions

1. Topological Spaces

Definition 1.1   A topological space is an ordered pair $(X,\mathcal{T})$, where $X$ is a set and $\mathcal{T}$ is a collection of subsets of $X$, satisfying the following axioms:

  • $X$ and $\varnothing$ belong to $\mathcal{T}$.
  • Any arbitrary union of elements of $\mathcal{T}$ belongs to $\mathcal{T}$.
  • The intersection of any pair of elements of $\mathcal{T}$ belongs to $\mathcal{T}$.

The elements of $\mathcal{T}$ are called open sets and the collection $\mathcal{T}$ is called a topology on $X$

Example 1.1  

  • $2^X$ is a topology on $X$, which is called the discrete topology.
  • $\{X,\varnothing\}$ is also a topology on $X$, which is called the trivial topology.

Example 1.2  

  • $\mathcal{T}_f=\{U\subset X\colon\ X\setminus U\ \text{is finite}\}\cup\varnothing$ is a topology on $X$, which is called the cofinite topology.
  • $\mathcal{T}_c=\{U\subset X\colon\ X\setminus U\ \text{is countable}\}\cup\varnothing$ is also a topology on $X$, which is called the cocountable topology.

Definition 1.2   Suppose $\mathcal{T}$ and $\mathcal{T}’$ are two topologies on $X$. We say $\mathcal{T}$ is comparable with $\mathcal{T}’$ if either $\mathcal{T}\supset\mathcal{T}’$ or $\mathcal{T}\subset\mathcal{T}’$. If $\mathcal{T}\supset\mathcal{T}’$, we say $\mathcal{T}$ is finier than $\mathcal{T}’$. If $\mathcal{T}\subset\mathcal{T}’$, we say $\mathcal{T}$ is coarser than $\mathcal{T}’$.

2. Basis and Subbasis

Definition 2.1   A basis for a topology on $X$ is a collection $\mathcal{B}$ of subsets of $X$ satifying the following properties:

  • For each $x\in X$, there exists $B\in\mathcal{B}$ such that $x\in B$.
  • Given $B_1, B_2\in\mathcal{B}$, for each $x\in B_1\cap B_2$, there exists $B_3\in\mathcal{B}$ such that $x\in B_3\subset B_1\cap B_2$.

Proposition 2.1   If $\mathcal{B}$ is a basis for a topology on $X$, there is topology $\mathcal{T}_\mathcal{B}$ generated by $\mathcal{B}$, in the sense that $U\in X$ is said to be open if for each $x\in U$, there exists $B\in\mathcal{B}$, such that $x\in B\subset U$.

Proof.  

  • It is easy to show that $X$ and $\varnothing$ are elements of $\mathcal{T}$ by the defining condition of openness.
  • For an indexed collection $\{U_\alpha\}_{\alpha\in J}$ of elements of $\mathcal{T}$, there exists an index $\beta$ for each point $x\in \bigcup_{\alpha\in J} U_\alpha$, such that $x\in U_\beta$. Since $U_\beta\in\mathcal{T}$, there is an element $B$ of $\mathcal{B}$ such that $x\in B\subset U_\beta\subset \bigcup_{\alpha\in J} U_\alpha$ which means $\bigcup_{\alpha\in J} U_\alpha\in\mathcal{T}$.
  • Given $U_1, U_2\in\mathcal{T}$, for each point $x\in U_1\cap U_2$, there exist $B_1, B_2\in\mathcal{B}$ such that $x\in B_1\subset U_1$, $x\in B_2\subset U_2$. Then, there exists $B_3\in\mathcal{B}$ such that $B_3\subset B_1\cap B_2$, which means $x\in B_3\subset U_1\cap U_2$, therefore $U_1\cap U_2\in\mathcal{T}$. $\square$
$\mathcal{T}_\mathcal{B}$ is the collection of all unions of elements of $\mathcal{B}$, since for each point $x$ of an element $U$ of $\mathcal{T}_\mathcal{B}$, there is an basis element $B_x$ such that $x\in B_x\subset U$, then $U=\bigcup_{x\in U} B_x$, and the converse is obvious. So we can also generate the topology $\mathcal{T}_\mathcal{B}$ in this particular way. On the other hand, we can also obtain the basis for a given topology, as the following proposition says.

Proposition 2.2   Let $(X,\mathcal{T})$ be a topological space, $\mathcal{C}$ is a collection of open sets of $X$. If $U\in\mathcal{T}$ contains at least one element $C$ of $\mathcal{C}$ for each $x\in U$, such that $x\in C$, then $\mathcal{C}$ is a basis for $\mathcal{T}$.

Proof.  

  • Since $X$ is an open set of itself, for each point $x\in X$ there is an element $C$ of $\mathcal{C}$, such that $x\in C$.
  • Given $C_1,C_2\in\mathcal{C}$, since $C_1$ and $C_2$ are open, then $C_1\cap C_2$ is open, which means for each $x\in C_1\cap C_2$, there is an element $C_3$ of $\mathcal{C}$ such that $x\in C_3\subset C_1\cap C_2$.\
    Thus $\mathcal{C}$ is a basis for a topology. Since $\mathcal{T}_\mathcal{C}$ is the collection of all unions of elements of $\mathcal{C}$, we have $\mathcal{T}_\mathcal{C}\subset\mathcal{T}$, and the coverse holds guaranteed by how we define the collection $\mathcal{c}$ exactly, therefore, $\mathcal{T}_\mathcal{C}=\mathcal{T}$. $\square$

Here are some examples of topologies generated by basis.

Example 2.1  

  • Let $\mathcal{B}$ be the collection of all open interval $(a,b)$ on $\mathbb{R}$. The topology generated by $\mathcal{B}$ is called a standard topology on $\mathbb{R}$.
  • Let $\mathcal{B}’$ be the collection of all half-open interval $[a,b)$ on $\mathbb{R}$.The topology generated by $\mathcal{B}’$ is called a lower limit topology on $\mathbb{R}$. The corresponding space is denoted by $\mathbb{R}_l$.
  • Let $K=\{\frac{1}{n}\colon\ n\in\mathbb{N}\}$, $\mathcal{B}’’$ be the collection of all sets of the form $(a,b)\setminus K$ on $\mathbb{R}$. The topology generated by $\mathcal{B}’’$ is called a K-topology on $\mathbb{R}$, The corresponding space is denoted by $\mathbb{R}_K$.

Proposition 2.3   Let $\mathcal{B}$ and $\mathcal{B}’$ be bases for the topologies $\mathcal{T}$ and $\mathcal{T}’$ on $X$ respectively, then the following are equivalent:

  1. $\mathcal{T}’$ is finer than $\mathcal{T}$.
  2. For each $x\in X$ and basis element $B\in\mathcal{B}$ containing $x$, there is a basis element $B’\in\mathcal{B}’$, such that $x\in B’\subset B$.

Proof.  

  • ($\Rightarrow$).   If $\mathcal{T}’$ is finer than $\mathcal{T}$, then $\mathcal{T}\subset\mathcal{T}’$, which means for each $x\in X$, each $B\in \mathcal{B}$ containing $x$ is an element of $\mathcal{T}’$, so that there is a basis element $B’\in\mathcal{B}’$ such that $B\subset B’$.
  • ($\Leftarrow$).   Let $U\in\mathcal{T}$, then $U=\bigcup_{x\in U} B_x$, where $B_x\in\mathcal{B}$ is a basis element containing $x$. Since there exists a basis element $B_x’\in\mathcal{T}’$ for each $x$ such that $B_x’\subset B_x$, then $U=\bigcup_{x\in U} B_x’$, which means $U\in\mathcal{T}’$. $\square$

Example 2.2   The topologies of $\mathbb{R}_l$ and $\mathbb{R}_K$ are strictly finer than the standard topology on $\mathbb{R}$, but are not comparable with one another.

Definition 2.2   A subbasis $\mathcal{S}$ for a topology on $X$ is a collection of subsets of $X$ whose union equals $X$.

One can easily check that all of the finite intersections of elements of $\mathcal{S}$ form a basis for a topology $\mathcal{T}$ on $X$, then $\mathcal{T}$ can be generated by the $\mathcal{S}$ by defining $\mathcal{T}$ as a collection of all the finite intersections of $\mathcal{S}$.

Exercises

Find a homeomorphism $\varphi:\mathbb{R}^n\to\mathbb{R}^n$ satisfies:

  • $\varphi=1_{\mathbb{R}^n\setminus\mathring{D^n}}$.
  • $\varphi(0)=p$, $p\in\mathring{D^n}$.

Where $D^n$ denotes the set $\{x\in\mathbb{R}^n\colon\ |x|\leq 1\}$.

Suppose that $\varphi\colon S^1 \to S^1$ is a homeomorphism. Prove that there exists a homeomorphism $\psi\colon D^2 \to D^2$, satisfies $\psi|_{S^1}=\varphi$.

(Hopf Link)
$$L=\{(x,y)\in\mathbb{C}^2\colon\ x^2+y^2=0, |x|^2+|y|^2=1\}$$

  1. Prove that $L\cong S^1\sqcup S^1$.
  2. Describe the image of the stereographic projection of $L$.
solution

  1. For $(x,y)\in L$，we have
    \begin{gather}
    x^2+y^2 =0 \implies (y-ix)(y+ix)=0 \implies y=\pm{i}x \\
    |x|^2+|y|^2 =1 \implies |x|=|y|=\frac{\sqrt{2}}{2}
    \end{gather}
    Let:
    $$
    U =\{(x,ix)\in\mathbb{C}^2\colon\ |x|=\frac{\sqrt{2}}{2}\},\quad
    V =\{(x,-ix)\in\mathbb{C}^2\colon\ |x|=\frac{\sqrt{2}}{2}\}
    $$
    then $L=U\sqcup V$ and we can simply construct homeomorphisms $(x,ix)\mapsto (\sqrt{2}\operatorname{Re}(x),\sqrt{2}\operatorname{Im}(x))$, and $(x,-ix)\mapsto (\sqrt{2}\operatorname{Re}(x),\sqrt{2}\operatorname{Im}(x))$ from $U\to S^1$ and $V\to S^1$ respectively, so we have the conclusion $L=U\sqcup V \cong S^1\sqcup S^1$.

  2. Consider the unit 3-sphere $\{(x,y)\in\mathbb{C}^2\colon\ |x|^2+|y|^2=1\}$, the stereographic projection $P$ of the sphere from the point $(0,1)$ maps each point $(x,y)$ differs from $(0,1)$ of the sphere to the point of intersection of the real line joining $(x,y)$ and $(0,1)$ with the 3-space $\{(p,q)\in\mathbb{C}^2\colon\ \operatorname{Re}(q)=0\}$. Expicitly, when $\operatorname{Re}(y)\neq 1$:
    $$
    P\colon\quad (x,y)\mapsto (\frac{\operatorname{Re}(x)}{1-\operatorname{Re}(y)},\frac{\operatorname{Im}(x)}{1-\operatorname{Re}(y)},\frac{\operatorname{Im}(y)}{1-\operatorname{Re}(y)})
    $$
    Since for each points $(x,y)\in L$, $\operatorname{Re}(y)\neq 1$, we can find the image for $U$ and $V$ separately from the formula above:
    $$
    P(U)=
    \begin{cases}
    u=w\\
    2u^2+(v+1)^2=2
    \end{cases}\quad\quad
    P(V)=
    \begin{cases}
    u=-w\\
    2u^2+(v-1)^2=2
    \end{cases}
    $$
    Equivalently:
    $$
    P(U)=
    \begin{cases}
    u=w\\
    u^2+(v+1)^2+w^2=2
    \end{cases}\quad\quad
    P(V)=
    \begin{cases}
    u=-w\\
    u^2+(v-1)^2+w^2=2
    \end{cases}
    $$
    where $P(U)$ and $P(V)$ are two circles on two intersecting planes (in this case the two planes are perpendicular), and both two centers are located on the line of the intersection of the two planes. Consider the radii and the distance between the centers, the two circles are going to be linked together, in the sense that neither can be continuously shrunk to a point without pcissing through the other one (see below).

If $(X,d)$ is a metric space and $A$ is a non-empty closed set of $X$, prove that $f(x)=\inf_{a\in A}\{d(x,a)\}$ is a continuous function on $X$ and also, $f(x)=0$ if and only if $x\in A$.

Let $\mathbb{Z}$ denotes the set of integers, $U\subseteq\mathbb{Z}$ is said to be symmetrical if $n\in U$ and $-n\in U$ are equivalent for $\forall n\in \mathbb{Z}$. We define a topology $\mathcal{T}_{\mathbb{Z}}$ on $\mathbb{Z}$, that the set of $\mathbb{Z}$ is open if it is symmetrical.

  1. Prove that $\mathcal{T}_{\mathbb{Z}}$ is actually a topology.
  2. If $A=\{-1,0,1,2\}\subseteq \mathbb{Z}$, determine $\mathring{A}$, $\bar{A}$ and $A’$.
  3. Is $A$ an open set on $\mathbb{Z}$? Or is $A$ closed?

Prove that $(\mathbb{R}^{n+1}\setminus\{0\})\cong (S^n\times\mathbb{R}_{>0})$, where $S^n=\{x\in\mathbb{R}^{n+1}\colon\ |x|=1\}$.

(Zariski Topology)   Subset $\mathbb{Z}$ of $\mathbb{C}^n$ is said to be Zariski closed if there exists polynomials $\{P_i\}_{i\in I}$ in $\mathbb{C}[x_1,x_2,\dots,x_n]$, such that $Z=\{z\in\mathbb{C}^n\colon\ P_i(z)=0, \forall i\in I\}$. If the complement of $U\subseteq \mathbb{C}^n$ is Zariski closed, we say $U$ is Zariski open. Prove the following:

  1. All Zariski open sets in $\mathbb{C}^n$ forms a topology on $\mathbb{C}^n$, which is called Zariski topology.
  2. $U_f=\{x\in\mathbb{C}^n\colon f(x)\neq{0}\}$ where $f\in\mathbb{C}[x_1,x_2,\dots,x_n]$, forms a topology basis of Zariski topology on $\mathbb{C}^n$.
  3. Suppose that $f_1,f_2,\dots,f_m$ are polynomials of $\mathbb{C}[x_1,x_2,\dots,x_n]$. Prove the map:
    $$(x_1,x_2,\dots,x_n)\mapsto(f_1(x),f_2(x),\dots,f_m(x))$$
    is continuous if we consider the Zariski topology on both $\mathbb{C}^n$ and $\mathbb{C}^m$.
